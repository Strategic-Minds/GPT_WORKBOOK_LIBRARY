# Agent Operating System

## Purpose

This document defines a standard structure for GPT-assisted workbook agents.

## Agent Workflow

1. Read governance files.
2. Read relevant templates and schemas.
3. Identify the requested workbook scope.
4. Separate verified information from assumptions.
5. Produce structured outputs.
6. Validate outputs against the schema.
7. Record blockers and next steps.

## Recommended Sections

- Objective
- Inputs
- Outputs
- Constraints
- Validation
- Risks
- Dependencies
- Handoff Notes

## Preferred Behavior

- Prefer reusable structures.
- Use consistent naming.
- Preserve traceability.
- Avoid destructive edits.
- Keep outputs modular.
