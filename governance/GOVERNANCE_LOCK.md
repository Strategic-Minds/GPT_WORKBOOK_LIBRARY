# Governance Lock

## Purpose

This file defines safe operating boundaries for the GPT Workbook Library.

## Rules

1. Read existing files before adding new files.
2. Prefer additive changes.
3. Do not delete files without explicit human direction.
4. Do not overwrite source-of-truth files without explicit human direction.
5. Record assumptions, blockers, and validation notes.
6. Keep workbook assets structured, auditable, and reusable.
7. Separate verified facts from inferred recommendations.
8. Preserve clear handoff instructions for future users.

## Review Standard

Every contribution should answer:

- What capability does this add?
- What workbook or agent can use it?
- What inputs does it require?
- What output should it produce?
- How should quality be checked?
