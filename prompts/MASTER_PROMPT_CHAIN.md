# Master Prompt Chain

## Purpose

Reusable prompt-chain patterns for workbook generation and GPT-assisted workflows.

## Prompt Pattern

1. Define the objective.
2. Define constraints.
3. Define required outputs.
4. Define validation requirements.
5. Define formatting standards.
6. Define continuation instructions.

## Workbook Generation Prompt

Create a workbook structure with:

- dashboard
- inputs
- calculations
- metrics
- simulations
- logs
- automation opportunities
- governance notes
- QA checklist

## Agent Handoff Prompt

Summarize:

- completed work
- blockers
- assumptions
- next exact step
