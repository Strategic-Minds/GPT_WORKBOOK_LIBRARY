# GPT Capability Map

## Purpose

This file helps GPTs and human operators locate reusable capabilities inside the GPT Workbook Library.

## Capability Groups

| Capability Group | Primary Location | Use Case |
|---|---|---|
| Workbook Planning | `templates/` | Create structured workbooks |
| Prompt Design | `prompts/` | Build reusable prompt chains |
| Skill Reference | `library/` | Find GPT skill patterns |
| Agent Support | `agents/` | Define assisted workflows |
| Governance | `governance/` | Review safe operating boundaries |
| Architecture | `docs/` | Understand system doctrine |
| Data Inventory | `data/` | Locate workbook lists and indexes |
| Simulation Planning | `simulations/` | Design forecast and scoring models |
| Workflow Support | `automation/` | Plan workflow routing and handoffs |

## How To Use

1. Identify the task type.
2. Match the task to a capability group.
3. Read the primary file in that group.
4. Use the related template or prompt.
5. Validate the output.
6. Record blockers and next steps.

## Priority Capabilities

- Build a workbook
- Build a prompt chain
- Build a skill card
- Build a tool card
- Build a workflow map
- Build a simulation model
- Build a quality checklist
- Build a handoff summary

## Capability Card Format

Each capability should include:

- name
- purpose
- location
- input requirements
- output format
- validation checklist
- related workbook
- related prompt
